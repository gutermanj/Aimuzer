= Aimuzer

Final Project | Music Platform Powered By Taste Recognition

=== Framework & Technologies
Rails 4, PostgreSQL, JavaScript, Ajax, CSS3, HTML5, SoundCloud API



=====Aimuzer uses a query system to use tags (Think of tags like hashtags that a user clicks on attached to a track)

=====The database is queried to find similar tracks based on tags selected


User Ability | Upload tracks, like tracks, create playlists, share tracks, follow users
